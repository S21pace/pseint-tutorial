¿Qué es PSeInt?

PSeInt es una herramienta educativa gratuita que ayuda a los estudiantes a dar sus primeros pasos en la construcción de algoritmos. Utiliza un pseudolenguaje sencillo en español para centrarse en los conceptos fundamentales de la algoritmia sin preocuparse por la sintaxis de un lenguaje de programación concreto. Con un entorno de edición intuitivo y ayudas integradas, PSeInt facilita la escritura de algoritmos y permite interpretar (ejecutar) el pseudocódigo de forma interactiva.

El pseudocódigo se utiliza como primer contacto para comprender estructuras de control, expresiones y variables, eliminando obstáculos propios de otros lenguajes. En este tutorial aprenderás los conceptos básicos de PSeInt de manera interactiva y dinámica.

Instalación de PSeInt

A continuación se describen los pasos básicos para instalar PSeInt en los sistemas operativos de escritorio más comunes. Estos procedimientos provienen de la página oficial de PSeInt y están resumidos aquí para que cualquiera pueda empezar rápidamente. Recuerda descargar la última versión estable desde la sección de descargas de SourceForge.

Windows

  1. Visita la página oficial de PSeInt y descarga el instalador para Windows.
  2. Ejecuta el archivo .exe descargado y sigue las instrucciones del asistente.
  3. Si prefieres una versión portátil, descarga el paquete comprimido, descomprímelo y ejecuta wxPSeInt.exe desde la carpeta extraída.

GNU/Linux

  1. Descarga el paquete .tgz adecuado a tu arquitectura. Si no sabes cuál elegir, ejecuta uname -m en una terminal; si el resultado es x86_64 utiliza la versión de 64 bits, en caso contrario la de 32 bits.
  2. Descomprime el paquete con tar -xzvf archivo.tgz y accede al directorio pseint.
  3. Ejecuta el archivo pseint: 
    cd pseint
./pseint
    La primera vez que lo ejecutes se te ofrecerá crear accesos directos en el menú y en el escritorio.
  4. Si la aplicación muestra mensajes sobre bibliotecas faltantes, instálalas con el gestor de paquetes de tu distribución.

macOS

  1. Descarga el archivo comprimido para macOS desde la página oficial.
  2. Descomprime el fichero haciendo doble clic; obtendrás la aplicación PSeInt.
  3. Para instalarla entre las aplicaciones del sistema, arrastra el icono de PSeInt a la carpeta Aplicaciones en Finder.
  4. En la primera ejecución puede aparecer un mensaje de seguridad. Haz clic con el botón derecho sobre la aplicación, selecciona Abrir y confirma la ejecución en el cuadro de diálogo.

Variables, Entrada/Salida y Operaciones

En PSeInt puedes definir variables utilizando la instrucción Definir nombre Como Tipo, donde el tipo puede ser Entero, Real, Lógico o Cadena. Para solicitar datos al usuario empleamos Leer, y para mostrar información se usa Escribir.

La asignación de valores se realiza con la flecha <-, que indica que el resultado de una expresión se almacena en la variable de la izquierda. PSeInt soporta operaciones aritméticas básicas como suma (+), resta (−), multiplicación (*), división (/) y módulo (%).

Ejemplo interactivo: Suma de dos números

Observa el siguiente algoritmo. Introduce dos números y presiona «Ejecutar» para ver el resultado. También puedes ejecutar el algoritmo paso a paso. 

Algoritmo SumaDosNumeros
    Definir a, b, suma Como Entero
    Escribir "Ingrese primer número:"
    Leer a
    Escribir "Ingrese segundo número:"
    Leer b
    suma <- a + b
    Escribir "La suma es: ", suma
FinAlgoritmo

Estructuras Condicionales

Las estructuras condicionales permiten que un algoritmo tome decisiones. En PSeInt se utiliza la instrucción Si condición Entonces ... Sino ... FinSi para evaluar una condición y ejecutar un bloque de instrucciones u otro.

La condición es una expresión lógica (verdadera o falsa). Por ejemplo, para comprobar si un número es par se puede evaluar si el resto de dividirlo por 2 es igual a cero.

Ejemplo interactivo: Número Par o Impar

Algoritmo ParOImpar
    Definir n Como Entero
    Escribir "Ingrese un número:"
    Leer n
    Si n % 2 = 0 Entonces
        Escribir "El número es par"
    Sino
        Escribir "El número es impar"
    FinSi
FinAlgoritmo

Bucles y Repetición

Los bucles o estructuras de repetición permiten ejecutar un conjunto de instrucciones varias veces. En PSeInt existen tres estructuras básicas: Para (un número conocido de iteraciones), Mientras (repite mientras se cumpla una condición) y Repetir ... Hasta (repite al menos una vez, hasta que la condición sea verdadera).

Ejemplo interactivo: Suma de 1 hasta N (bucle Para)

Algoritmo Sumar1HastaN
    Definir n, i, suma Como Entero
    Escribir "Ingrese un número positivo:"
    Leer n
    suma <- 0
    Para i <- 1 Hasta n Con Paso 1 Hacer
        suma <- suma + i
    FinPara
    Escribir "La suma desde 1 hasta ", n, " es ", suma
FinAlgoritmo

Ejercicio: Completa el pseudocódigo del factorial

Rellena los espacios para construir un algoritmo que calcule el factorial de un número (n!).

Algoritmo Factorial
     n, i, fact Como Entero
    Escribir "Ingrese un número:" 
     n
    fact <- 1
    Para i <- 1 Hasta n Con Paso 1 Hacer
        fact <- fact * i
    
     "El factorial de ", n, " es ", fact
FinAlgoritmo

Práctica adicional

Ahora que conoces las bases, intenta resolver los siguientes problemas por tu cuenta. Escribe tu pseudocódigo en el cuadro y luego compáralo con una solución sugerida.

1. Calcular el área de un rectángulo

Escribe un algoritmo que solicite la base y la altura de un rectángulo y muestre su área.

Algoritmo AreaRectangulo
    Definir base, altura, area Como Real
    Escribir "Ingrese la base:" 
    Leer base
    Escribir "Ingrese la altura:" 
    Leer altura
    area <- base * altura
    Escribir "El área del rectángulo es:", area
FinAlgoritmo

2. Contador de números positivos y negativos

Solicita diez números al usuario e indica cuántos son positivos y cuántos negativos.

Algoritmo ContarPositivosNegativos
    Definir i, num, positivos, negativos Como Entero
    positivos <- 0
    negativos <- 0
    Para i <- 1 Hasta 10 Hacer
        Escribir "Ingrese un número:" 
        Leer num
        Si num > 0 Entonces
            positivos <- positivos + 1
        Sino
            Si num < 0 Entonces
                negativos <- negativos + 1
            FinSi
        FinSi
    FinPara
    Escribir "Cantidad de positivos: ", positivos
    Escribir "Cantidad de negativos: ", negativos
FinAlgoritmo
